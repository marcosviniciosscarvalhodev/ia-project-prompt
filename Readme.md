Modos de Interação do Copiloto

O Copiloto oferece diferentes modos para atender objetivos específicos durante o desenvolvimento.

❓ Ask

Utilizado para tirar dúvidas e entender o código sem realizar alterações. Ideal para explicações, análise de erros e conceitos.

✏️ Edit

Serve para modificar código existente, como refatorações, correções de bugs e melhorias de lógica.

🧭 Plan

Planeja a solução antes da implementação, dividindo o problema em etapas e explicando a estratégia antes de codar.

🤖 Agent

Modo mais autônomo. Pode navegar pelo projeto, criar e modificar arquivos e executar tarefas complexas envolvendo múltiplos componentes.

📚 Study

Focado no aprendizado. Ensina conceitos, estimula o raciocínio e guia o usuário durante a resolução do problema.

Resumo
Ask → Entender o código.
Edit → Alterar o código.
Plan → Planejar antes de implementar.
Agent → Executar tarefas complexas com autonomia.
Study → Aprender enquanto desenvolve.
